# Carpeta de imágenes del Homebanking

Imágenes agregadas para el diseño visual Alfin. Están en `public/assets/homebanking/` para que React pueda cargarlas con rutas públicas sin tocar imports, hooks, servicios, backend ni rutas.

## Archivos colocados

- `alfin-logo.png`: logo principal usado por `src/components/ui/Logo.jsx`.
- `hero-homebanking.png`: imagen grande del landing.
- `credito-aprobado.png`: bloque visual de crédito/préstamo.
- `login-seguro.png`: imagen lateral del login.
- `dashboard-cliente.png`: imagen del dashboard del cliente.
- `transferencias.png`: bloque visual de transferencias.
- `pagos-cuotas.png`: bloque visual de pagos/cuotas.
- `metricas-cliente.png`: bloque visual de métricas/control.
- `resumen-cliente.png`: bloque visual del resumen financiero.

## Importante

No se cambiaron rutas del proyecto ni conexiones al backend. Las páginas ya apuntaban a estas rutas públicas, solo se agregaron los archivos y se conectó el componente Logo al PNG.
